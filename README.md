# Ananya Infocard

A small, responsive Next.js portfolio (infocard) that showcases contact details and key links.

Built with React and Tailwind CSS - minimal and easy to customize.

Quick start
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build for production: `npm run build`

Where to look
- `app/` - Next.js app routes and pages
- `components/` - React components (card, logo, flipping card)
- `public/` - static images and icons
- `app/global.css` - global styles and color tokens

Notes
- If you change the color palette, update `app/global.css` and any inline SVG fills in `components/`.

Simple, focused, and ready to customize.
This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
